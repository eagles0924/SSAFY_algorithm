arr = [5, 1, 4, 2, 8]

# 수정한 코드
n = len(arr)
for i in range(n-1, 0, -1):     # i의 범위 수정
    for j in range(i):
        if arr[j] > arr[j+1]:   # 부등호 방향 수정
            arr[j], arr[j+1] = arr[j+1], arr[j]     # ==를 =로 수정

1. i의 범위 수정
문제에서 버블 소트를 이용하여 리스트를 정렬하고자 한다. 버블 소트는 리스트에서 양옆의 원소들 값을 비교하여 정렬하는 방법이다.
j의 범위가 range(i)로 고정되었으므로 리스트의 뒤에서 앞 방향으로 정렬을 진행한다. 따라서 i는 정방향이 아닌 역방향이 되어야 한다.
arr의 인덱스는 0부터 n-1까지 존재하므로 start는 n-1로, j의 범위를 고려했을 때 종료 인덱스는 1이면 되기 때문에 end는 0으로 설정한다.
방향은 역방향이므로 -1로 설정한다.

2. 부등호 방향 수정
인덱스가 j와 j+1인 두 원소의 값을 비교할 때, 오름차순으로 정리해야 하기 때문에 작은 인덱스인 j의 원소값이 j+1보다 클 때 정렬을 실행해야 한다.

3. 연산자 수정
== 연산자는 논리 연산자로 두 값이 서로 같은지 아닌지를 판단한다. 버블 소트에서는 두 원소의 위치를 서로 바꿔주는 작업을 진행하기 때문에
논리 연산자가 아닌 =를 사용해야 한다.
